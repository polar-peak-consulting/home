Course Description

This course covers the basics of Kubernetes, including container orchestration, deployment, scaling, and management.

Course Modules

  • Module 1: Introduction to Kubernetes
  • Module 2: Setting Up Kubernetes Cluster
  • Module 3: Kubernetes Pods
  • Module 4: Kubernetes Deployments
  • Module 5: Kubernetes Services
  • Module 6: Kubernetes Persistent Volumes and Persistent Volume Claims
  • Module 7: Kubernetes Ingress
  • Module 8: Monitoring and Logging in Kubernetes
  • Module 9: Kubernetes Best Practices and Advanced Topics

Module 0: Understanding Containerization

This module provides an overview of Kubernetes, covering:

  • Overview of containerization
  • Introduction to Docker
  • Docker basics: images, containers, Dockerfile
  • Benefits of containerization

Module 1: Introduction to Kubernetes

This module provides an overview of Kubernetes, covering:

  • What is Kubernetes?
  • Why use Kubernetes?
  • Kubernetes architecture overview
  • Kubernetes features and benefits

Module 2: Setting Up Kubernetes Cluster

This module provides an overview of Kubernetes, covering:

  • Installing Kubernetes (local setup using Minikube or cloud-based setup using managed Kubernetes services)
  • Configuring Kubernetes cluster
  • Interacting with Kubernetes using kubectl

Module 3: Kubernetes Pods

This module provides an overview of Kubernetes, covering:

  • Understanding Pods
  • Creating and managing Pods
  • Pod lifecycle
  • Multi-container Pods

Module 4: Kubernetes Deployments

This module provides an overview of Kubernetes, covering:

  • Introduction to Deployments
  • Creating and managing Deployments
  • Rolling updates and rollbacks
  • Scaling Deployments

Module 5: Kubernetes Services

This module provides an overview of Kubernetes, covering:

  • Introduction to Services
  • Types of Services (ClusterIP, NodePort, LoadBalancer, ExternalName)
  • Exposing applications with Services

Module 6: Kubernetes Persistent Volumes and Persistent Volume Claims

This module provides an overview of Kubernetes, covering:

  • Storage classes

This module provides an overview of Kubernetes, covering:

  • Introduction to Ingress
  • Configuring Ingress resources

  • Routing traffic to Kubernetes services

  • Logging with filebeat and Elasticsearch
